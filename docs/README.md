# Documentation Index

This directory contains the repository documentation for reusable workflows and catalog usage rules.

## Available documents

- `docs/usage.md`: how consumers should reference and use reusable workflows
- `docs/versioning.md`: catalog semantic versioning policy
- `docs/ci-nodejs.md`: Node.js reusable CI workflow contract and examples

## Writing rule

All documentation in this repository should be written in English.

This includes:

- README files
- Workflow-specific documentation
- Examples
- Comments embedded in code snippets
